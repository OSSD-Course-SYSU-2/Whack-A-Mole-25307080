项目简介
本项目为鸿蒙课程作业，使用 ArkTS + Stage 模型 开发的多合一休闲小游戏合集，满足一次开发多段部署。
tablet和手机上均可使用。整体界面风格统一，全部游戏页面增设顶部退出按钮，可随时返回游戏大厅。
开发语言：ArkTS
开发工具：DevEco Studio
应用模型：Stage 模型
运行设备：鸿蒙模拟器、鸿蒙真机
公共通用功能
玩家昵称登录，未登录无法进入游戏
独立游戏规则说明页面
各游戏页面统一配置「返回大厅」按钮，退出自动清空定时器，防止后台占用
每款游戏独立记录历史最高得分
完整页面链路：首页→登录→规则→游戏对局→结算
各游戏详细功能
1. 打地鼠
   总计 3 个递进关卡，关卡越高地鼠刷新频率越快
   三种可选锤子，拥有不同得分倍率
   连击机制：连续击打获得额外加分，漏掉地鼠连击清零
   单关需要达到满分 80% 分数方可晋级
   支持游戏暂停、本局主动退出
   游戏结束后展示本局得分与通关结果
2. 颜色找茬（共计 10 关）
   难度梯度设计，关卡越高网格越大、颜色差值越小
   1~3 关：4×4 网格，色差较大，入门难度
   4~6 关：5×5 网格，色差降低
   7~10 关：6×6 网格，色差极小，高难度
   每一关仅有一个异色方块，点击正确方块进入下一关
   全部 10 关通关后进行总分结算并记录最高分
3. 射击打靶
   红色靶标自动从右向左持续移动
   点击靶标即可累计得分
   任意靶标移出左边界，立刻判定游戏结束
   对局支持一键退出，结算页面可保存最高分
4. 记忆翻牌
   提供三种棋盘规格：4×4 / 6×6 / 8×8
   翻开两张相同数字卡牌即可消除卡牌
   统计本局翻牌总次数，记录每种规格下的历史最优成绩
   8×8 超大棋盘使用滚动布局，界面不会溢出，搭配顶部固定退出按钮
   项目技术亮点
   使用@Builder封装复用页面组件，代码结构清晰
   @State装饰器管理页面响应式状态
   合理使用 Grid、Scroll、Column、Row 等布局，适配手机竖屏
   定时器统一管理，页面销毁时主动清除，优化性能
   模块化拆分各个游戏，后期便于新增游戏
项目文件结构
entry/src/main/ets
   └── entryability
   └── pages
   └── index.ets  主入口页面（包含全部游戏逻辑与Builder）
运行说明
   使用 DevEco Studio 导入项目
   启动模拟器或连接鸿蒙真机
   运行项目，输入昵称完成登录
   选择任意游戏，阅读规则后开始游玩
   所有游戏页面顶部均可点击返回大厅随时退出
