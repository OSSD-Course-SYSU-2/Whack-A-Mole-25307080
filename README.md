鸿蒙打地鼠游戏 - Lab1 项目



项目简介

&#x09;本项目是基于HarmonyOS开发的打地鼠小游戏。游戏实现了地鼠随机生成、点击计分、连击加成、难度递进等核心玩法，界面简洁友好，交互流畅。



开发环境

&#x09;开发工具：DevEco Studio 

&#x09;开发语言：ArkTS

&#x09;目标API版本：API 11

&#x09;项目构建工具：hvigor



核心功能

&#x09;1. 三关难度递进：

&#x20;  	 	第一关：地鼠刷新慢、停留时间长，适合新手熟悉操作

&#x20;   		第二关：地鼠刷新速度加快，停留时间缩短，难度提升

&#x20;   		第三关：地鼠刷新频繁，还会出现“炸弹”干扰项，考验反应力

&#x09;2.  计分系统：

&#x20;   		普通地鼠：+10分

&#x20;   		连击加成：连续击中可获得额外分数奖励

&#x20;   		错误惩罚：点击炸弹会扣除分数或中断连击

&#x09;3.  完整游戏流程：

&#x20;   		开始界面 → 游戏界面 → 结算界面

&#x20;   		显示当前关卡、分数、连击数、倒计时信息

&#x09;4.  基础UI与交互：

&#x20;   		点击反馈动画、地鼠出现/消失动画

&#x20;   		游戏暂停、重新开始功能
